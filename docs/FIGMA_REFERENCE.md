## Objetivo

Construir una Home real para LICAN EVENTS con estética oscura, premium, underground y profesional, orientada a:

1. Vender entradas.
2. Captar clientes para servicios de producción.
3. Captar artistas, DJs y productores.
4. Generar base de datos mediante newsletter.
5. Posicionar LICAN EVENTS como promotora y productora cultural de música electrónica en Tarragona y Barcelona.

## Menú definitivo

El menú principal debe ser:

- Inicio
- Eventos
- Servicios
- Artistas
- Galería
- Contacto

Botón destacado en el header:

- Entradas

No incluir “Colaboradores” en el menú principal.
No incluir “Nuestras Fiestas” en el menú principal.
Las fiestas deben estar dentro de la sección Eventos.

## Estilo visual

La web debe mantener:

- fondo negro
- degradados púrpura
- azul eléctrico
- cian
- blanco
- glow controlado
- tarjetas oscuras
- bordes finos
- tipografía contundente
- estética club
- sensación audiovisual
- alto contraste
- movimiento sutil

Evitar:

- estética gaming barata
- exceso de rayos
- datos inventados
- imágenes externas
- marcas falsas
- colores demasiado saturados
- diseño corporativo frío

## Assets

Usar únicamente assets locales:

- /public/images
- /public/videos
- /public/logos

No usar imágenes externas.

## Reglas técnicas

- No modificar /dist.
- No usar imágenes externas.
- No crear datos falsos.
- No romper el proyecto existente.
- Crear componentes reutilizables.
- Separar datos en archivos dentro de /src/data.
- Mantener código responsive.
- Priorizar rendimiento.
- Preparar estructura para futuras páginas.
