# LICAN EVENTS - Design Guide

## 1. Dirección visual
La identidad visual de LICAN EVENTS debe ser oscura, premium, underground y de alto impacto. La estética debe transmitir modernidad, energía y exclusividad.

## 2. Paleta visual
- Fondo oscuro profundo
- Púrpura como color principal de identidad
- Azul eléctrico para contraste y energía
- Cian neón para acentos y detalles de movimiento
- Alto contraste para legibilidad y fuerza visual

## 3. Estilo de interfaces
- Diseño limpio, editorial y premium
- Tipografía fuerte y reconocible
- Jerarquía visual clara
- Espacios amplios y composición impactante
- Elementos visuales con presencia dramática

## 4. Uso de imagen
- Priorizar fotografía real de eventos y experiencias
- Mostrar público, cabina, humo, luces, DJs y planos potentes
- Evitar imágenes borrosas, demasiado oscuras o poco nítidas
- Mantener coherencia visual entre todas las secciones
- Cargar todas las imágenes desde public/images

## 5. Aplicación por sección
- Home: impacto inmediato, identidad y propuesta de valor
- Eventos: narrativa de experiencias y calendario
- Servicios: profesionalismo, confianza y claridad
- Nuestras Fiestas: energía, tensión y emoción
- Colaboradores: autoridad, calidad y alianza de marcas
- Lanza tu Carrera: apertura, futuro y oportunidades
- Galería: narrativa visual de alto impacto
- Contacto: claridad, confianza y acción

## 6. Principios de diseño
- El diseño debe sentirse premium sin ser excesivamente ornamental.
- La experiencia visual debe ser potente, elegante y memorable.
- Cada sección debe trabajar como una pieza de marca y no como un bloque genérico.
